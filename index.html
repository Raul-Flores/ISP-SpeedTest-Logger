Um die Leistung und Stabilität meiner LTE Internetanbindung von Drei zu protokollieren, habe ich ein kleines Software Setup zur Erfassung und Speicherung von Geschwindigkeits / Ping Tests erstellt.

Ein shell script wird stündlich auf meiner NAS ausgeführt und speichert die Ergebnisse der Speedtests in ein JSON Logfile. Diese Daten werden genutzt um die nachfolgenden Graphen zu erstellen.

Mehr Info findet Ihr auf meinem GitHub Repository
